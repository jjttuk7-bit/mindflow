## 1. 콘텐츠 캡처

Mindflow는 4가지 형식의 콘텐츠를 지원합니다.

### 텍스트 (아이디어)
- 하단 입력창에 텍스트를 입력하고 저장
- 짧은 메모, 아이디어, 생각 등 자유롭게 기록
- Shift+Enter로 줄바꿈, Enter 또는 저장 버튼으로 저장

### 링크
- URL을 입력하면 자동으로 링크 타입으로 인식
- Open Graph 메타데이터(제목, 설명, 파비콘) 자동 추출
- 카드에 미리보기 형태로 표시

### 이미지
- 카메라 촬영 또는 갤러리에서 업로드
- 드래그 앤 드롭 지원 (데스크톱)
- AI가 이미지 내용을 한국어로 자동 설명

### 음성 메모
- 마이크 버튼으로 녹음 시작/중지
- Whisper AI로 한국어 음성 자동 전사
- 녹음 파일과 텍스트 모두 저장

## 2. AI 자동 처리

콘텐츠 저장 시 AI가 자동으로 수행하는 작업:

### 태그 자동 생성
- 콘텐츠 내용을 분석하여 1~3개 태그 생성
- 기존 태그를 우선 재사용하여 일관성 유지
- 콘텐츠 타입별 최적화된 태깅 (텍스트/링크/이미지/음성)

### 요약 자동 생성
- 100자 이상의 콘텐츠에 대해 한 줄 요약 생성
- 피드 카드와 검색 결과에서 요약 표시

### 이미지 캡션
- 업로드된 이미지의 내용을 AI가 한국어로 설명
- 스크린샷의 텍스트 추출
- 비동기 처리: 저장 후 백그라운드에서 캡션 생성

### 임베딩 생성
- 모든 콘텐츠에 대해 768차원 벡터 임베딩 생성
- 시맨틱 검색과 "잊혀진 기억 되살리기" 기능에 활용

### 프로젝트 자동 분류 (Pro)
- AI가 콘텐츠를 분석하여 적절한 프로젝트에 자동 배정
- 기존 프로젝트가 3개 이상일 때 활성화

## 3. 피드 & 카드

### 피드 뷰
- **리스트 뷰**: 카드 형태의 기본 레이아웃
- **타임라인 뷰**: 날짜별 시간순 정렬
- **정렬**: 최신순 / 오래된순 / 타입별

### 카드 기능
각 카드에서 사용 가능한 액션:

| 액션 | 설명 |
|------|------|
| 📌 고정 | 피드 상단에 고정 |
| 📁 프로젝트 이동 | 프로젝트에 수동 배정 |
| ✏️ 편집 | 텍스트, 이미지 캡션, 음성 전사 수정 |
| 🔗 공유 | 공유 가능한 링크 생성 |
| 📦 아카이브 | 피드에서 숨김 (복원 가능) |
| 🗑️ 삭제 | 영구 삭제 |
| 🔍 관련 항목 | 유사한 콘텐츠 미리보기 (호버 시) |

### 모바일 제스처

모바일 환경에서 카드를 터치하여 빠르게 조작할 수 있습니다.

#### 스와이프 제스처

| 제스처 | 동작 | 시각 피드백 |
|--------|------|-------------|
| **← 좌측 스와이프** | 아카이브 (피드에서 숨김) | 빨간 배경 + Archive 아이콘 |
| **→ 우측 스와이프** | 고정/해제 | 노란 배경 + Pin 아이콘 |

- 터치 후 **80px 이상** 밀어야 동작이 실행됩니다
- 80px 도달 전: 아이콘이 반투명으로 미리보기 표시
- 80px 도달 후: 아이콘이 완전 불투명으로 전환 (실행 확정)
- 손가락을 떼면 카드가 자동으로 원위치로 돌아옵니다
- 휴지통 내 카드에서는 스와이프가 비활성화됩니다

#### 방향 감지

- 터치 시작 후 **8px 이상** 이동하면 가로/세로 방향을 자동 판별합니다
- **가로 이동 > 세로 이동** → 스와이프로 처리 (스크롤 차단)
- **세로 이동 > 가로 이동** → 일반 스크롤로 처리

#### 롱프레스 (길게 누르기)

- 카드를 **0.5초 이상** 누르면 컨텍스트 메뉴가 열립니다
- 메뉴 항목: 고정, 아카이브, 삭제, 공유, 프로젝트 이동
- 롱프레스 도중 손가락이 8px 이상 움직이면 자동 취소됩니다

### 콘텐츠 타입별 표시
- **텍스트**: 요약 + 전체 내용 토글
- **링크**: OG 미리보기 카드 (제목, 설명, 파비콘)
- **이미지**: 썸네일 + 라이트박스
- **음성**: 오디오 플레이어 + 전사 텍스트 + 재생시간

## 4. 검색

`Ctrl+K` (또는 `Cmd+K`)로 검색 다이얼로그 열기

### 키워드 검색
- 제목, 내용, 태그에서 즉시 텍스트 매칭
- 실시간 결과 표시

### 시맨틱 검색
- AI 임베딩 기반 의미 유사도 검색
- "비슷한 의미"의 콘텐츠를 찾아줌
- 일치율(%) 표시
- 예: "여행 계획"을 검색하면 "제주도 맛집 리스트"도 찾아줌

## 5. 프로젝트 & 태그

### 프로젝트
- 사이드바에서 프로젝트 생성/관리
- 컬러 인디케이터로 시각적 구분
- 카드에서 직접 프로젝트 이동 가능
- 프로젝트별 아이템 수 표시
- 프로젝트 선택 시 해당 아이템만 필터링

### 태그 (컬렉션)
- AI가 자동 생성한 태그로 콘텐츠 분류
- 사이드바에서 태그 목록 확인
- 태그 이름 변경 및 삭제
- 태그 클릭 시 해당 태그의 아이템만 필터링
- 사용 빈도순 정렬

## 8. AI 채팅

저장된 지식을 기반으로 AI와 대화:

### 사용 방법
- **데스크톱**: 우측 패널에서 채팅
- **모바일**: 하단 네비게이션 "Chat" 탭

### 기능
- AI가 저장된 아이템을 검색하여 답변
- 참조한 아이템 출처 표시
- 채팅 세션 히스토리 관리
- 새 채팅 생성 및 이전 채팅 불러오기

### 예시 질문
- "지난주에 저장한 여행 관련 내용 알려줘"
- "프로젝트 A에 대해 정리해줘"
- "최근 읽은 기사들의 공통 주제는?"

## 9. 잊혀진 기억 되살리기

Mindflow만의 핵심 기능:

### 작동 방식
1. 최근 저장한 메모의 임베딩을 분석
2. 7일 이상 된 과거 메모 중 유사한 것을 발견
3. 피드 상단에 앰버 색상 카드로 표시

### 표시 정보
- 기간 (예: "15일 전의 기억", "2달 전의 기억")
- 과거 메모의 내용과 태그
- 최근 메모와의 연결 관계

### 특징
- 세션당 1회 표시 (X 버튼으로 닫기 가능)
- 다음 세션에서 새로운 기억을 보여줌
- 유사 메모가 없으면 랜덤으로 오래된 메모 제공

## 10. 월간 인사이트

`/insights` 페이지에서 활동 분석:

### 무료 기능
- 총 캡처 수 (히어로 통계)
- 타입별 활동 (막대 차트)
- 소스별 활동 (파이 차트: 웹/텔레그램/API)
- 일일 활동 히트맵
- 상위 프로젝트
- 할 일 완료 진행률
- 미확인 링크 수, 지연된 할 일, 오래된 핀 알림

### Pro 기능
- **관심사 분석**: 상위 토픽, 트렌딩 태그
- **월간 다이제스트**: AI 요약, 핵심 인사이트

## 11. 공유 & 내보내기

### 공유
- 개별 아이템에 대한 공유 링크 생성
- 토큰 기반의 공개 페이지 (인증 불필요)
- 타입별 색상, 태그, 요약 표시

### 내보내기
- **Markdown (.md)**: 마크다운 형식으로 다운로드
- **JSON (.json)**: 구조화된 데이터 형식
- **AI 요약**: 간략/상세 요약 리포트 생성
- 현재 필터(타입, 태그, 프로젝트) 반영

## 12. 텔레그램 연동

### 설정
1. 설정 페이지에서 "텔레그램 연결" 클릭
2. 생성된 링크로 텔레그램 봇에 접속
3. `/start` 명령으로 계정 연결

### 캡처 가능 콘텐츠
- 텍스트 메시지 → 텍스트 아이템
- URL → 링크 아이템 (OG 메타데이터 포함)
- 사진 → 이미지 아이템 (AI 캡션 포함)
- 음성 메시지 → 음성 아이템 (자동 전사)

### 봇 명령어
| 명령어 | 설명 |
|--------|------|
| `/search 키워드` | 저장된 지식에서 검색 |
| `/recent` | 최근 저장 아이템 보기 |
| `/todo` | 할 일 목록 보기 |
