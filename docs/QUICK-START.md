# Mindflow 퀵 스타트 가이드

> 5분 안에 Mindflow를 시작하는 방법

---

## Step 1: 회원가입 & 로그인

1. Mindflow 앱에 접속
2. 이메일과 비밀번호로 회원가입
3. 이메일 인증 후 로그인

**모바일 팁**: 브라우저 메뉴에서 "홈 화면에 추가"를 선택하면 앱처럼 사용할 수 있습니다.

---

## Step 2: 첫 번째 메모 저장하기

하단 입력창에서 바로 시작:

```
💡 텍스트: "오늘 회의에서 나온 아이디어 - 사용자 온보딩 개선"
🔗 링크:  "https://interesting-article.com"
📷 이미지: 카메라/갤러리 버튼 클릭
🎤 음성:  마이크 버튼으로 녹음
```

저장하면 AI가 자동으로:
- ✅ 태그 생성 (1~3개)
- ✅ 요약 생성 (긴 텍스트)
- ✅ 이미지 설명 생성
- ✅ 음성 텍스트 변환

---

## Step 3: 콘텐츠 정리하기

### 프로젝트 만들기
1. 사이드바에서 "Projects" 섹션 확인
2. `+` 버튼으로 새 프로젝트 생성
3. 카드의 📁 버튼으로 아이템을 프로젝트에 배정

### 중요한 메모 고정
- 카드에서 📌 버튼 클릭
- 또는 모바일에서 우측으로 스와이프
- 고정된 메모는 피드 최상단에 표시

### 필요 없는 메모 정리
- 카드에서 아카이브 버튼 클릭
- 또는 모바일에서 좌측으로 스와이프
- 아카이브된 메모는 "아카이브" 폴더에서 복원 가능

---

## Step 4: 검색으로 찾기

`Ctrl+K` (또는 `Cmd+K`)로 검색창 열기:

- **키워드 검색**: 정확한 단어로 빠르게 찾기
- **시맨틱 검색**: "비슷한 의미"로 찾기 (AI 기반)

예시:
- 키워드: "회의 아이디어" → 정확히 그 단어가 포함된 메모
- 시맨틱: "업무 개선" → 관련된 모든 메모 (다른 표현도 포함)

---

## Step 5: AI와 대화하기

저장한 지식을 기반으로 AI에게 질문:

1. 데스크톱: 우측 채팅 패널 / 모바일: Chat 탭
2. 질문 입력

```
"지난주에 저장한 내용 중 마케팅 관련된 거 정리해줘"
"프로젝트 X에 대해 요약해줘"
"최근 북마크한 기사들의 핵심 포인트는?"
```

AI가 저장된 아이템을 참조하여 답변하고, 출처를 함께 표시합니다.

---

## Step 6: 잊혀진 기억 만나기

별도의 설정 없이 자동으로 작동합니다:

- 피드를 열면 상단에 과거 메모가 가끔 나타남
- AI가 최근 메모와 관련된 오래된 메모를 찾아줌
- "15일 전의 기억", "2달 전의 기억" 등으로 표시
- X 버튼으로 닫을 수 있고, 다음 세션에서 새로운 기억 표시

---

## 보너스: 텔레그램 연동

어디서든 빠르게 캡처:

1. 설정 → 텔레그램 연결 클릭
2. 텔레그램 봇 링크 열기
3. `/start`로 계정 연결
4. 이후 봇에 메시지/사진/음성 전송 → 자동 저장

---

## 핵심 제스처 요약 (모바일)

| 제스처 | 동작 |
|--------|------|
| 좌측 스와이프 | 아카이브 |
| 우측 스와이프 | 고정/해제 |
| 길게 누르기 | 컨텍스트 메뉴 |
| 아래로 당기기 | 새로고침 |

---

## 핵심 단축키 요약 (데스크톱)

| 단축키 | 동작 |
|--------|------|
| `Ctrl+K` | 검색 |
| `Enter` | 저장 |
| `Shift+Enter` | 줄바꿈 |
| `Ctrl+Enter` | 편집 저장 |
| `Escape` | 닫기 |
