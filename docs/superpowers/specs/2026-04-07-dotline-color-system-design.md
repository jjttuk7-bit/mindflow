# DotLine 컬러 시스템 재디자인 스펙

**날짜:** 2026-04-07
**상태:** 승인됨
**컨셉:** 미도리 저널 (라이트) × 숲속 밤 (다크)

---

## 배경 및 동기

기존 DotLine의 "Warm Editorial" 컬러 시스템은 방향은 맞지만 생동감이 부족하고 AI가 만든 느낌이 났다. 사람들이 노트를 사고, 펜슬을 고르고, 기록 자체에서 즐거움을 느끼던 아날로그 시대의 물성과 온기를 디지털 인터페이스에 담는 것이 목표다.

**3가지 원칙:**
1. 자연 소재의 이름을 가진 색 — 잉크, 종이, 이끼, 가죽, 세피아
2. AI가 고른 선명하고 차가운 색 배제
3. 기존 코드의 CSS 변수 이름 유지, 값만 교체

---

## 컨셉: 미도리 저널

일본 트래블러스 노트(Midori/Traveler's Notebook)에서 영감. 깔끔하고 절제되어 있지만 손의 온기가 남아있는 미니멀리즘. 오프화이트 종이 위 차콜 먹물, 미도리 그린 강조색.

## 콘텐츠 유형별 잉크 컬러

기존 `terracotta`, `sage`, `dusty-rose`, `amber-accent` 커스텀 색상 교체.

| 유형 | 이름 | 라이트 값 | 다크 값 | 메타포 |
|------|------|-----------|---------|--------|
| 아이디어 (text) | 잉크 블루 | `#3A5C8A` | `#5A7AB0` | 만년필 파랑 잉크 |
| 링크 (link) | 미도리 그린 | `#2A6651` | `#3A7A5E` | 트래블러스 노트 표지 |
| 이미지 (image) | 번트 시에나 | `#8B4A2A` | `#A86040` | 세피아 사진 톤 |
| 음성 (voice) | 앰버 브라운 | `#7A5A2A` | `#8B6A2A` | 오래된 테이프 레코더 |

```css
/* 교체 예시 */
--color-terracotta: oklch(0.42 0.08 258);   /* 잉크 블루 — 아이디어 */
--color-sage:       oklch(0.465 0.095 162); /* 미도리 그린 — 링크 */
--color-dusty-rose: oklch(0.42 0.095 45);   /* 번트 시에나 — 이미지 */
--color-amber-accent: oklch(0.42 0.07 65);  /* 앰버 브라운 — 음성 */
```

## 타이포그래피

기존 `DM Serif Display` (헤드라인) + `DM Sans` (본문) 폰트 유지. 세리프의 손글씨 감성과 산세리프의 가독성이 미도리 저널 컨셉과 잘 어울림.

---

## 카드 디자인 방향

- 카드 왼쪽에 콘텐츠 유형 잉크 컬러 3px border-left 적용
- 태그 칩: filled 대신 **아웃라인 스타일** (border + 잉크 컬러 텍스트)
- 라이트: `border-radius: 4-5px` (기존 12px에서 축소 — 더 종이 느낌)
- 다크: 카드 배경이 배경보다 살짝 초록빛으로 밝음

---

## 구현 범위

1. `app/globals.css` — CSS 변수값 교체 (라이트/다크/OLED 블랙 3개 섹션)
2. 커스텀 컬러 토큰 (`warm-*`, `terracotta`, `sage` 등) 값 업데이트
3. 카드 border-radius 조정 (선택적)
4. 태그 칩 스타일 아웃라인으로 변경 (선택적)

**변경하지 않는 것:** CSS 변수 이름, 폰트, 레이아웃, 컴포넌트 구조

## OLED 블랙 모드

기존 `.black` 클래스 유지. 배경은 순수 `#000000`, 카드는 극도로 어두운 숲 톤 `oklch(0.06 0.008 155)` 적용. 프라이머리는 다크 모드와 동일한 `#3A7A5E` 사용.
